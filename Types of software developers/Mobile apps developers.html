Job Description:

A mobile app developer uses programming languages and development skills to create, test, and develop applications on mobile devices. They work in popular operating system environments like iOS and Android and often take into account UI and UX principles when creating applications.


Salary

$71,396 avg per yr according to Glassdoor in Edmonton, Alberta


Requirement/Prequisites for job:

One must possess software development skills and knowledge; you can develop these skills by earning an associate's or bachelor's degree in computer science or a similar discipline
