Job Description:

A desktop developer are programmers who write code for software applications that run on operating systems like macOS, Windows, and Linux, and don't need to be connected to the internet.



Salary:

$54,769 avg per yr according to Glassdoor in Edmonton, Alberta



Requirement/Prequisites for job:

You need to be able to code, Employers also look for detail-oriented and patient programmers. desktop developers need to be curious. no specific degree path for desktop developers. Many job listings in the field ask only for a four-year degree or that vague qualification: "equivalent experience." Most employers are looking for candidates with a bachelor's degree in computer science or master's degree in computer science, engineering, mathematics, or a related discipline
